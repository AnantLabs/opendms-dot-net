

A message from a client to the server is a request.
A message from the server to a client is a response.

Request
Headers - we add a header "X-OpenDMS-Meta-Length" which holds a numeric value indicating the length of the json package
Json package
	type - the Type of the object
	timestamp - the date/time of the request
	conversation_id - a guid
Byte package

Response
Same as request

Messages.MessageBase - base for all Requests and Responses
Messages.JsonOnlyBase - base for all Requests and Responses (2nd level)
